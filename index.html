Change the title

To change the captcha title, simply click on the name at the top and start typing.

Change the image

To change the image, have one ready in your clipboard. Paste the image using CTRL + V.

Change the correct pattern

If you'd like to change the correct pattern, select all of the tiles that you want to be correct. This is by default none. Once you have selected them all, press the Retry button in the bottom-left of the captcha to save the selection. Upon completing the captcha, it will disappear.

Save the image

When finished, you can save the captcha to your clipboard. To do this, press the Headphones button in the bottom-left of the captcha. The image will be saved to your clipboard.
